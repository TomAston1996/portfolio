
# Personal Portfolio

My personal portfolio website. A portfolio built in React and NextJS. Simple, clean and fast.

## Run Project 

Run this project with Yarn

```bash 
  yarn && yarn run dev
```
    
## Deployment

To deploy this project run

```bash
  yarn build
```
